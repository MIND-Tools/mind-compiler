################################################################################
#   Mind Lifecycle example: OVERVIEW
################################################################################

This example shows how to declare a life-cycle controller in the ADL, how to 
implement it in a primitive component and how to wrap an application with a 
life-cycle controller in a bootstrap that will initialize it correctly.


################################################################################
#   LAUNCHING THE BASE EXAMPLE WITH MAKE
################################################################################
From the command line, simply type:
 $ make simpleLCC
  
  
Afterwards, execute the binary "simpleLCC" file with:
 $ build/simpleLCC
